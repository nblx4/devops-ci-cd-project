# DevOps CI/CD Project

## Project Overview
This project demonstrates a CI/CD pipeline using GitHub Actions and Node.js.

## Features
- Automated build pipeline
- Automated unit testing
- Artifact upload
- CI/CD workflow automation
- npm dependency caching

## Technologies Used
- Node.js
- Express.js
- GitHub
- GitHub Actions

## Pipeline Workflow
1. Developer pushes code to GitHub
2. GitHub Actions triggers automatically
3. Dependencies are installed
4. Unit tests are executed
5. Artifacts are uploaded

## Pipeline Stages

### Build Pipeline
- Install dependencies
- Run unit tests
- Upload artifacts

## Output
The pipeline automatically validates the project whenever code is pushed.
